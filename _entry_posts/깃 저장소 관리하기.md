# 이력 비우기

현재 저장소의 이력을 비우는 기능은 없으므로, 이력이 없는 저장소를 만들어 이주해야 한다.

~~~
git checkout --orphan NEW_BRANCH_NAME
~~~

- `--orphan` 스위치로 텅 빈 브랜치를 만든다.
- `add` `commit` `push`를 수행한다.
- 이제 필요에 따라 기존 브랜치를 삭제할 수 있다.

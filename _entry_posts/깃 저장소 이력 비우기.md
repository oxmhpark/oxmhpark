1. 기존 브랜치로부터 이력이 없는 브랜치를 생성한다.\
   `git checkout --orphan NEW_BRANCH_NAME`
2. 새 브랜치에 커밋한다.
   `git add .`
   `git commit -m 'cleanup'`
3. 기존 브랜치를 삭제한다.\
   `git branch -d CURRENT_BRANCH_NAME`
4. 새로 만든 브랜치의 이름을 변경한다.\
   `git branch -m CURRENT_BRANCH_NAME`
